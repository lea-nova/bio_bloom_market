
# Bio Bloom Market 


## Installation du projet
Dépendance PHP : 
```
composer install
```
Installer npm : 
```
npm install 
```


**1.** Lancer le serveur de développement
Serveur symfony
 
```
symfony server:start
 ```

Lancer Webpack Encore pour mettre à jour les changements en temps réels.
```
npm run watch
```
